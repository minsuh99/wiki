# W2M5 - Word Cloud 작동 방법

## 1. 텍스트 전처리 (Processing Text)
입력받은 긴 텍스트를 토큰으로 쪼개고 전처리하는 과정
- 정규식을 활용한 토큰화
- 소유격('s) 표현 제거
- 숫자 제거
- 짧은 단어 필터링
- 불용어 제거

## 2. 빈도 계산
전처리된 토큰들을 바탕으로 단어별 등장 횟수를 집계하는 과정
- 단어의 빈도를 기준으로 내림차순 정렬 후 상위 200개(기본값) 단어 추출
- 제일 많이 나온 단어의 빈도를 1로 정규화

## 3. 폰트 크기 설정
단어들의 빈도 순위와 빈도가 차지하는 비율을 적절히 조정하여 폰트 크기를 결정함

## 4. 레이아웃 계산 (=단어들의 배치 계산)
위에서 결정된 폰트 크기의 단어를 캔버스에서 겹치지 않도록 배치하는 과정
- 빈도가 높은 단어부터 배치
- `IntegralOccupancyMap`를 활용해 겹치지 않는 빈 공간 탐색
- 단어를 회전시켜서도 배치할 수 있으면 배치 -> 안 되면 폰트 크기를 줄여서라도 배치

## 5. 출력
단어 위치, 색상, 방향 등을 결정하는 단계
- 특정 mask 형태로 워드클라우드 변환 가능
